
MicroHelp

MicroHelp is a micro-task exchange platform designed to connect users who need small, quick tasks done with helpers who can complete them efficiently. Built for hackathons and MVP demos, it features credits, task time estimation, helper reviews, and AI-powered task suggestions.

Features

Public Task Board – Browse tasks without login for live demos.

Post Tasks – Create tasks with title, description, category, estimated time, and credit reward.

Task Feed – View all available tasks, filtered by category, reward, and time.

Task Detail – View detailed task info, accept tasks, and mark them completed.

Dashboard – Track posted tasks, completed tasks, earned/spent credits, and reputation.

Credit System – Posters spend credits to post tasks; helpers earn credits when completing tasks.

Time Estimation System – Estimated task completion times help manage expectations.

Helper Review System – Rate helpers after task completion; reputation score calculated.

AI Integration – Suggests task categories, estimated time, and helps auto-generate descriptions.

Responsive UI – Built with React + Vite and Tailwind CSS.

Backend API – Node.js + Express with MongoDB Atlas via Mongoose.

Tech Stack

Frontend: React + Vite, Tailwind CSS, React Router

Backend: Node.js, Express, Mongoose

Database: MongoDB Atlas

AI Integration: OpenAI API (optional)
